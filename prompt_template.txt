
Crie um modelo de template html com importação do jquery e Bootstrap via CDN, que tenha uma navbar fixa no topo do site, um conteúdo que ocupe todo o espaço disponível do meio do site e um rodapé no final do template. Essa navbar deve ter os links centralizados e conter algum dropdown. Também quero no final dela (alinhado no end) um botão para login ou cadastro, que depois será um dropdown para o usuário autenticado com opção de logout e outras coisas. O template deve ser responsivo, moderno, minimalista. Gosto bastante da cor azul petróleo. Evite ficar criando css personalizado e dê preferência por usar as classes do bootstrap para formatar a estrutura. Além disso, crie uma paleta de cores com cores alternativas para success, primary, info, danger, warning, com cores modernas e que combinem com o azul petróleo. Crie também os elementos com essa paleta de cores, como botões, alertas, cores de link, texto, bordas, cores de fundo e outras coisas mais semelhantes aos elementos do Bootstrap. Seja criativo e moderno.
