📘 Sobre o Sistema

Este sistema foi desenvolvido com o objetivo de gerenciar cursos, alunos, professores e matrículas de forma organizada e eficiente.

O sistema permite cadastrar cursos, associar disciplinas, criar turmas, vincular professores e realizar matrículas de alunos.

Ele foi desenvolvido utilizando Django e Bootstrap, seguindo boas práticas de desenvolvimento web.
